# 图片资源文件夹

本文件夹需要包含以下图片资源：

## 必需图片

1. `tarot-card.png` - 主要展示的塔罗牌图片（建议尺寸：300x500px）
2. `avatar.png` - 用户默认头像（建议尺寸：64x64px）
3. `reader-1.png` 到 `reader-6.png` - 六个不同风格的塔罗师头像（建议尺寸：200x200px）

## 图片要求

- 所有图片建议使用PNG格式，支持透明背景
- 塔罗师头像建议使用圆形设计
- 塔罗牌图片建议使用具有神秘感的卡牌设计

## 临时占位图片

在正式图片资源准备好之前，可以使用以下在线工具生成占位图片：

1. 访问 https://placeholder.com/ 或 https://placehold.co/
2. 按照上述建议尺寸生成占位图片
3. 下载并重命名为相应的文件名

例如，生成300x500px的塔罗牌占位图：https://placehold.co/300x500/purple/white?text=Tarot+Card

## 图片来源建议

- 可使用免费图片库如Unsplash、Pexels或Pixabay获取符合项目风格的图片
- 使用AI图像生成工具创建风格统一的塔罗师头像
- 确保所有使用的图片资源拥有合适的使用许可
